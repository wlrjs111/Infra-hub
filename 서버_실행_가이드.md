======================================================
  Infra-Hub 서버 실행 가이드 (git pull 이후)
======================================================

전제: 이미 아래는 완료된 상태
  cd /home/mzadmin
  git clone https://github.com/wlrjs111/Infra-hub.git
  cd Infra-hub
  git pull


------------------------------------------------------
1. 실행 권한 부여
------------------------------------------------------

  chmod +x setup_v98.sh
  chmod +x collect_asset_info.sh


------------------------------------------------------
2. Infra-Eye 설치 (장애 감지 + 알람)
------------------------------------------------------

  * 이 서버에 처음 설치하는 경우에만 실행
  * 이미 설치된 서버는 건너뛰기

  sudo bash setup_v98.sh

  실행 중 아래 4가지를 입력하라는 창이 뜸:

    WEBHOOK_URL    구글챗 웹훅 URL
    SHEETS_URL     Apps Script 웹 앱 URL (/exec 로 끝남)
    CLIENT_NAME    고객사명 (예: proxmox)
    MANAGER_NAME   담당자 이름

  입력한 값은 /etc/telegraf/.env 에 저장됨.
  collect_asset_info.sh 도 이 값을 그대로 재사용함.

  설치 중간에 HW 상태 확인 후 SEL 초기화 여부(Y/N)를 물어봄.
  현재 하드웨어 이상 없는지 확인한 뒤 Y 입력.
  (나중에 따로 초기화: sudo ipmitool sel clear)


------------------------------------------------------
3. Infra-Data 자산 정보 수집
------------------------------------------------------

  sudo bash collect_asset_info.sh

  OS, CPU, 메모리, 디스크, 백업설정, 설치된 애플리케이션,
  실행중인 서비스 정보를 수집해서 구글 시트(infra-data 탭)로 전송.

  같은 호스트에서 재실행하면 새 행이 아니라
  기존 행이 갱신됨 (덮어쓰기).


------------------------------------------------------
4. 정상 반영 확인
------------------------------------------------------

  Infra-Eye  : 구글챗에 배포 완료 메시지 도착 확인
  Infra-Data : Infra-Hub 대시보드 -> Infra-Data 탭에서
               해당 호스트 행 확인


------------------------------------------------------
요약 (외우기용 3줄)
------------------------------------------------------

  chmod +x setup_v98.sh collect_asset_info.sh
  sudo bash setup_v98.sh          (신규 서버만)
  sudo bash collect_asset_info.sh
